- **dragon-ball 死点击(P1)**: 8/42 会话角色选择阶段多次 dead click。
- **VIP/credit 门禁 bug(P0)**: `fusion/generate` 对 VIP 不豁免 credit<1 拦截，webhook 失败则付费用户被 402。
- **/profile 死链(P1)**: header.tsx:103、mobile-nav.tsx:94 链到不存在的 /profile → 404。
- **auth-debug 页(P1)**: app/auth-debug/page.tsx 暴露在 production。
- **死配置(P2)**: config/subscriptions.ts (Starter $11/Business $29/Enterprise $99) 未被定价页引用，与硬编码价格冲突。
- **入口分裂(P2)**: 首页 CTA→/dragon-ball，定价页"Start Creating Free"→/ai，但 /ai 样本 0 触达。
